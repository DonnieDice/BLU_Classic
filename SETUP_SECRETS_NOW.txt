IMPORTANT: You need to set the secrets for CurseForge uploads to work!

Since I cannot access your secret values, you need to run these commands yourself.
When prompted, enter the SAME tokens you use in your BLU addon.

OPTION 1: Interactive (Recommended)
Run each command and paste the token when prompted:

gh secret set CF_API_KEY --repo DonnieDice/BLU_Classic
gh secret set WAGO_API_TOKEN --repo DonnieDice/BLU_Classic  
gh secret set WOWI_API_TOKEN --repo DonnieDice/BLU_Classic

OPTION 2: From Files
If you have the tokens saved in text files:

type cf_key.txt | gh secret set CF_API_KEY --repo DonnieDice/BLU_Classic
type wago_token.txt | gh secret set WAGO_API_TOKEN --repo DonnieDice/BLU_Classic
type wowi_token.txt | gh secret set WOWI_API_TOKEN --repo DonnieDice/BLU_Classic

OPTION 3: Direct (Less Secure - tokens visible in history)
Replace YOUR_TOKEN with actual values:

echo YOUR_CF_KEY | gh secret set CF_API_KEY --repo DonnieDice/BLU_Classic
echo YOUR_WAGO_TOKEN | gh secret set WAGO_API_TOKEN --repo DonnieDice/BLU_Classic
echo YOUR_WOWI_TOKEN | gh secret set WOWI_API_TOKEN --repo DonnieDice/BLU_Classic

WHERE TO FIND YOUR TOKENS:
- If you don't remember them, check:
  1. Your password manager
  2. Browser saved passwords
  3. Local notes/files
  4. Or regenerate from the platforms:
     - CurseForge: https://authors.curseforge.com/account/api-tokens
     - Wago: https://addons.wago.io/account/apikeys
     - WoWInterface: https://www.wowinterface.com/downloads/filecpl.php?action=apitokens

After setting the secrets, verify with:
gh secret list --repo DonnieDice/BLU_Classic

You should see:
- CF_API_KEY
- WAGO_API_TOKEN
- WOWI_API_TOKEN
